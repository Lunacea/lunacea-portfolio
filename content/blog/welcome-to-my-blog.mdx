## コードの例

簡単なReactコンポーネントの例：

```jsx
import { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div className="counter">
      <h2>カウンター: {count}</h2>
      <button onClick={() => setCount(count + 1)}>
        クリック
      </button>
    </div>
  );
}

export default Counter;
```
